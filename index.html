O que é HTML e CSS

Curso completo de HTML e CSS do 0 à sua primeira pagina com apenas essas duas linguagens

Conhecendo as duas linguagens

HTML (HyperText Markup Language)

O HTML é a linguagem de marcação utilizada para estruturar o conteúdo de páginas da web. Ele organiza elementos como textos, imagens, vídeos e links, definindo o que será exibido e a hierarquia de informações na página. É considerado o "esqueleto" de um site.

CSS (Cascading Style Sheets)

O CSS é a linguagem usada para estilizar o conteúdo estruturado pelo HTML. Com ele, é possível definir cores, fontes, espaçamentos, alinhamentos e outros aspectos visuais de uma página. O CSS é o responsável por transformar o "esqueleto" do HTML em algo visualmente atrativo e funcional.

O que é HTML ?

O HTML organiza e estrutura o conteúdo das páginas da web usando tags como estas:

  • <h1> para titulos principais
  • <p> para paragrafos
  • <img> para usar imagens

O que é CSS ?

O CSS adiciona estilo ao conteúdo criado com HTML. Ele pode ser aplicado de três maneiras:

  • Inline: dentro do <HTML>
  • Interno: Em um bloco <style> no próprio HTML.
  • Externo: Em um arquivo separado <.css>
